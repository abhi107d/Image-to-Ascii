#	**Image to ascii converter**

*converts image's to ascii art

## Quick start

```CONSOLE

$ python image2ascii.py [IMAGE PATH]

```

## Demo 
 
```CONSOLE

$ python image2ascii.py image.png

```
*file will be saved as save.txt
